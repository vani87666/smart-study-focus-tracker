# 📘 Smart Study Focus Tracker

## 📌 Overview
The Smart Study Focus Tracker is a simple web-based application developed to help students monitor and improve their study habits. It calculates a focus score based on study time and break time, providing instant feedback to enhance productivity and discipline.

---

## 🎯 Objective
- To track study time and break time  
- To evaluate focus level using a simple method  
- To encourage better time management among students  
- To provide instant feedback for self-improvement  

---

## 💡 Features
- Input study time (in minutes)  
- Input break time (in minutes)  
- Automatic focus score calculation  
- Focus level classification (Low, Medium, High)  
- Warning for excessive breaks  
- Simple and user-friendly interface  

---

## ⚙️ Working Principle
The system uses a rule-based approach:

**Focus Score = Study Time – Break Time**

Based on the score:
- Low Focus (< 30)  
- Medium Focus (30–60)  
- High Focus (> 60)
